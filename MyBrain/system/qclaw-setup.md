## 起步前先核对 4 件事

| 事项 | 状态 | 备注 |
|---|---|---|
| 1. 微信账号 | 你应该有 | QClaw 用微信扫码登录绑定 |
| 2. macOS | ✅ 你有 | QClaw 支持 Mac（也支持 Win） |
| 3. 4000 万 token / 天免费额度 | ⚠️ 不确定 | 视频说有，但是否对海外 IP 给同样额度未知。装上看 |
| 4. 中国 IP / VPN | ⚠️ 可能需要 | 腾讯产品**有可能**对海外访问限速 / 限流。如果遇到下载或登录卡壳，挂回国 VPN |

**如果在第 1 步（下载）就卡住** → 跳到本文末尾"故障排除"。

## Step 1：下载 + 安装（≤5 分钟）

QClaw 入口（按可能性排序，逐个试）：

1. **腾讯官方下载页**：搜索 `qclaw 腾讯` 或访问 `qclaw.qq.com`
2. **腾讯电脑管家**：如果上面找不到独立下载，可能内嵌在电脑管家里
3. **应用宝 / 知乎专栏推荐链接**：上面的页面里有"立即下载"按钮，照点

下载到 `.dmg` 后双击挂载 → 拖到 Applications → 启动。

> ⚠️ macOS 第一次运行可能弹"无法验证开发者"。处理：System Settings → Privacy & Security → 滚到底，给 QClaw "Open Anyway"。

---

## Step 2：扫码绑定微信（30 秒）

启动后会出二维码。手机微信打开 → 扫一扫 → 同意授权。

绑定后客户端 UI 应该会显示你的微信头像 + 默认 agent 工作区。

---

## Step 3：第一次试 — 创建一个简单 agent（5 分钟）

按视频的流程：

1. 左侧聊天框 → 点 **新建 agent**
2. 选人设 / 自定义。**第一次建议自定义**，prompt 写得简单：

   ```
   你是一个简单的写作助手。
   帮我把零散的笔记整理成结构化的列表。
   语气客观直白，不要 emoji。
   ```

3. 名字随便起，比如"小整"
4. 点 **现在创作** → agent 创建完成

---

## Step 4：跑一个真实任务测能力（10 分钟）

挑一个**你已经做过 / 知道答案**的任务，让 agent 跑，对比它的结果跟你预期。这是评估工具能力最快的方法。

**建议任务（任选一个）**：

### A. 整理任务（最轻量）

把以下文字塞给小整 agent，看它能不能整理成列表：

```
今天搞定了 git 备份初始化（initial commit 了 117 个文件），
部署了 ai-watch 和 email-triage 两个 daemon skill，发现 Brain Corp
2026 实习窗口已关闭（4/1 下架），把 4 条 approvals 全部归档处理。
```

**期待**：3-4 条干净的列表。
**评估点**：会不会瞎加内容、会不会加 emoji、会不会编造你没说的事。

### B. 文件操作（中等）

让 agent 帮你把桌面某个 PDF 或图片移到指定文件夹（**先建一个测试文件夹避免误操作真东西**）。

**评估点**：
- 它真能操作本地文件吗？还是只是"看起来能"？
- 安全沙箱是否提示了"这是文件写操作"？

### C. Multi-Agent 测试（重点）

按视频说的，建第二个 agent（比如"小写"，人设：写作专家）。给它跟小整完全相同的任务，对比两者输出差异。

然后**真的让两个 agent 串接**：让小整整理一段文字 → 把整理结果交给小写润色。看协作真的"自动 handoff"还是你需要复制粘贴中转。

**这一步是检验视频"multi-agent 并行"是真功能还是营销话术的关键。**

## 故障排除

### 找不到下载入口

试试这些备用 query：
- `qclaw.qq.com`
- `腾讯 ai 龙虾`
- `腾讯 AI agent 桌面端`
- `腾讯 PC 管家 AI 助手`

最坏情况搜不到 → 可能 QClaw 还在内测 / 限地区开放。等几个月再试。

### 下载卡住 / 网速极慢

腾讯产品对海外 IP 有时限速。处理：
1. 挂中国 VPN（任何稳定的，不要免费的那种）
2. 或让国内朋友下载 .dmg 发你

### 微信扫码后返回"账号未授权"

可能是：
- 账号不在内测白名单 → 等开放或申请内测
- 海外手机号注册的微信账号优先级低 → 用国内手机号注册的微信号试

### 4000 万 token 额度看不到

实际"免费额度"展示位置可能在：
- 客户端右上角头像下拉菜单
- 设置 → 账户 → 用量
- 控制台 / 后台网页（链接通常在客户端"了解更多"按钮）

如果完全看不到 / 装上要付费才能用 → 视频信息可能过时 / 优惠已结束。**不要为试用付费**，直接卸。

## 退出 / 卸载

试完不想继续：

```
Applications → QClaw → 拖到废纸篓
~/Library/Application Support/QClaw/ → 删掉
~/Library/Preferences/com.tencent.qclaw.plist → 删掉
（在 Mac Finder 用 Shift+Cmd+G 输入路径访问）
```

vault 完全没动 → 不影响 Claude Code / daemon / 你之前所有的工作。
